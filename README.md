# Movie Recommender System

A content-based movie recommender system built using cosine similarity.

## Features
- Recommends top 5 similar movies
- Uses cosine similarity
- Fetches posters using TMDB API
- Deployed using Streamlit

## Dataset
Dataset source:
- Kaggle TMDB Movie Dataset
- https://www.themoviedb.org/

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- TMDB API
